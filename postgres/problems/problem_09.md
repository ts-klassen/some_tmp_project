# 09 ユーザ別注文数ランキング

**難易度**：★★  
**学習トピック**：SELECT, LEFT JOIN, サブクエリ, ORDER BY

## 説明
`orders` テーブルをユーザごとに集計して注文数を求め、
その結果を `users` テーブルと結合して、各ユーザの注文数をランキング形式で取得してください。

## 制約

* サブクエリで注文数を集計すること
* LEFT JOIN を使用すること
* ORDER BY で注文数の降順かつユーザIDの昇順を指定すること
* 出力カラムのカラム名は `user_id`, `user_name`, `order_count` とすること
