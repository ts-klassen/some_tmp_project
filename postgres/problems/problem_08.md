# 08 一度も注文されていない商品

**難易度**：★★  
**学習トピック**：EXISTS / NOT EXISTS

## 説明
`order_items` に一度も登場しない商品 (つまり、これまでに
まったく購入されていない商品) を抽出してください。

## 制約
* `NOT EXISTS` もしくは `LEFT JOIN … IS NULL` を使うこと
* 結果は `id` 昇順で並べること

## 想定出力例

| id | name           | price |
|----|----------------|-------|
|  9 | Rare Sticker   |   480 |
| 34 | VR Headset     | 76000 |
| …  | …             |   …   |
