# 07 ユーザ別注文数の集計

**難易度**：★  
**学習トピック**：SELECT, GROUP BY, COUNT

## 説明
`orders` テーブルからユーザ (`user_id`) ごとの注文数を集計してください。

## 制約
* GROUP BY を使用すること
* WHERE, HAVING, JOIN などの追加句は使用しないこと

## 想定出力例（先頭 5 行）

| user_id | order_count |
|---------|-------------|
| 1       | 3           |
| 2       | 3           |
| 3       | 3           |
| 4       | 3           |
| 5       | 3           |
