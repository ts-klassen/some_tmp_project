# 05 各注文の合計個数

**難易度**：★★  
**学習トピック**：JOIN, GROUP BY

## 説明
`order_items` テーブルを元に、**注文 ID (`order_id`) ごとの
合計数量 (`total_quantity`)** を求めてください。

## 制約
* 結果は `order_id` 昇順で並べること

## 想定出力例（先頭 5 行）

| order_id | total_quantity |
|----------|----------------|
|      100 |             12 |
|      101 |              1 |
| …        |            …   |
